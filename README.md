# Bingo Casino Royale

## Português 

### Sobre o Projeto 

O Bingo Casino Royale é um jogo de bingo online desenvolvido com PHP e JavaScript. Este projeto simula uma experiência simples de bingo em tempo real, com cartelas para múltiplos jogadores, sorteio automático de números, verificação de vencedores e interface temática de cassino de luxo. 

### Tecnologias Utilizadas 

* PHP
* HTML5
* CSS3
* Bootstrap 4
* jQuery
* Font Awesome 

### Funcionalidades 

* Geração aleatória de cartelas de bingo
* Sorteio automático de números
* Verificação automática de padrões vencedores (linhas, colunas e diagonais)
* Registro de vencedores
* Exibição de modal para anúncio de vencedores
* Interface temática de cassino de luxo
* Layout totalmente responsivo

### Estrutura de Arquivos 

* index.php - Página principal do jogo
* bingo.php - Lógica do jogo e funções principais
* data/players.php - Nomes dos jogadores e avatares
* data/drawn_numbers.txt -  Arquivo TXT para armazenar números já sorteados
* data/winners.txt - Arquivo TXT para registrar os vencedores
* css/style.css - Estilos e tema visual

## English

### About the Project

Bingo Casino Royale is an online bingo game developed with PHP and JavaScript. This project simulates a simple real-time bingo experience, with multiple player cards, automatic number draw, winner verification and a luxury casino themed interface.

### Technologies Used

* PHP
* HTML5
* CSS3
* Bootstrap 4
* jQuery
* Font Awesome

### Features

* Random bingo card generation
* Automatic number drawing
* Automatic verification of winning patterns (rows, columns and diagonals)
* Winners registration
* Modal display for winners announcement
* Luxury casino themed interface
* Fully responsive layout

### File Structure

* index.php - Game home page
* bingo.php - Game logic and main functions
* data/players.php - Player names and avatars
* data/drawn_numbers.txt - TXT file to store numbers already drawn
* data/winners.txt - TXT file to register winners
* css/style.css - Styles and visual theme
